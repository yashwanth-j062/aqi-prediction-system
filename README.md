# AQI Prediction System

Real-time Air Quality Index prediction system using Machine Learning models for 56 Indian cities.

## Features
- Hourly data collection from OpenWeather & IQAir APIs
- ML Models: Linear Regression, Random Forest, XGBoost, LSTM
- Live dashboard with real-time visualizations
- Health alerts based on AQI thresholds
- Deployed on Render (Backend + Frontend)

## Tech Stack
- **Backend**: Python (Flask), PostgreSQL
- **Frontend**: React.js, Chart.js
- **ML**: scikit-learn, XGBoost, TensorFlow/Keras
- **Deployment**: Render
