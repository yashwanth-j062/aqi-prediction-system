# AQI Prediction System - Frontend

React-based dashboard for real-time AQI visualization.

## Setup

### 1. Install Dependencies

## Features

- Live AQI metrics for 56 Indian cities
- Real-time charts and graphs
- Health alerts based on AQI levels
- 7-day forecast visualization
- Responsive design

- React 18
